EXAMINATION ATTENDANCE SYSTEMS

This is a group project titled "Examination Attendance System." The system is developed using Spring Boot as the framework. Maven is utilized as a tool for implementing the web service, serving both the provider and the consumer. For the consumer-side front-end, the webpage design incorporates various languages such as Thymeleaf, Hypertext Markup Language (HTML), and Cascading Style Sheet (CSS). The primary purpose of this web service is to manage examination attendance. It records student attendance and provides reports to the invigilator and lecturer regarding the student's status. The attendance can be recorded using four different input methods: self-check-in, QR code, matriculation number, and fingerprint. To build the web service, we have implemented various methods such as GET, POST, DELETE, and PUT. Additionally, we have tested the API using the POSTMAN software.
